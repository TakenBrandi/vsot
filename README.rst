Dev Setup
---------

1. Install antlr 4:

Ubuntu/Debian:

.. code-block:: console
    sudo apt-get install antlr4

Mac
.. code-block:: console
    brew install antlr

2. Install poetry

3. Dev installation

.. code-block:: console
    poetry install
